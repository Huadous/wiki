# Edition Zero: JSON Handling

## 定义
Edition Zero: JSON Handling 是一份专注于 Edition Zero 中 JSON 序列化与反序列化行为的设计文档。它讨论了在 Editions 框架下 JSON 映射的语义收敛、字段命名约定以及与 proto3 JSON 规范的差异，旨在指导 Edition Zero 的 JSON 规范制定。

## 关键特征
- 聚焦于 JSON 序列化与反序列化在 Edition Zero 中的语义收敛
- 定义字段命名约定，明确 proto 字段到 JSON 名称的映射规则
- 对比并阐明与 proto3 JSON 规范之间的差异
- 与 Edition Zero: Converged Semantics 密切相关，共同推动跨语言实现的语义统一
- 属于 Protobuf Editions 历史设计文档集合的一部分

## 应用
- 作为 Edition Zero 阶段 JSON 规范制定的设计依据
- 为不同语言实现提供 JSON 序列化与反序列化的统一语义参考
- 作为评估与 proto3 JSON 兼容性和差异性的对照参考
