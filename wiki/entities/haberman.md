# haberman

## 基本信息
- Type: person
- Source: [[sources/editions-protobuf-editions-design-features]]

## 描述
haberman 是 Protobuf Editions Features 设计提案的主要作者之一，以 GitHub 用户名 @haberman 出现。该提案于 2022 年 10 月 13 日获得批准，是 [[entities/Protobuf Editions|Protobuf Editions]] 项目下的核心技术设计文档。haberman 与 [[entities/fowles|fowles]] 共同提出使用自定义选项机制定义和表示特性的方案，取代了早期使用字符串定义特性的设计。该方案旨在利用 Protobuf 已有的自定义选项机制，避免引入新的语法形式，同时支持语言特定特性、继承、目标属性等高级机制。
