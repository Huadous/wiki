# protobuf.dev

## 基本信息
- Type: product
- Source: [[sources/editions-edition-zero-features]]

## 描述
protobuf.dev 是 Protocol Buffers 的官方网站与文档门户，承载语言指南、参考手册、Editions 特性文档等核心资源。该站点是 Protobuf 生态中规范文档与迁移指南的主要发布渠道。在 Editions 设计相关文档中，protobuf.dev 上的 [[sources/editions-edition-zero-features|Edition Zero Features]] 页面明确指出其内容将被 https://protobuf.dev/editions/features 页面（Feature Settings for Editions）取代，标志着旧版文档逐步向 Editions 体系迁移。此外，protobuf.dev 上发布的 Application Note: Field Presence 被引用为 [[concepts/presence-discipline|presence discipline]] 的进一步阅读材料，是理解字段存在性语义的重要参考资源。protobuf.dev 同时也是 [[entities/Protocol-Buffers|Protocol Buffers]] 官方信息的权威来源。
