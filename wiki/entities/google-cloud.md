# Google Cloud

## 基本信息
- Type: product
- Source: [[sources/overview|overview]]

## 描述
Google Cloud 是 Google 提供的云服务平台，涵盖计算、存储、网络、大数据和机器学习等领域的云计算服务。在 Protocol Buffers 官方文档的"谁在使用 Protocol Buffers"部分中，Google Cloud 被列为典型应用案例之一，表明 Protocol Buffers 在云基础设施组件中扮演重要角色。Google Cloud 与 [[entities/grpc|gRPC]] 和 [[entities/envoy-proxy|Envoy Proxy]] 等项目共同展示了 Protocol Buffers 在分布式系统中的广泛应用。
