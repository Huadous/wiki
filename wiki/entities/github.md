# GitHub

## 基本信息
- Type: organization
- Source: [[sources/overview|overview]]

## 描述
GitHub 是一家基于云端的软件开发平台提供商，负责托管大量开源项目的源代码。在 Protocol Buffers 生态系统中，GitHub 扮演着核心托管角色：[[entities/protocol-buffers|protocol-buffers]] 的源代码和官方文档均由 GitHub 托管。谷歌官方支持的部分语言的运行时库（runtime libraries）位于 GitHub 仓库中，而非直接在谷歌仓库内维护。此外，第三方开发者对 Protocol Buffers 的非官方语言支持也通过独立的 GitHub 项目实现。Protocol Buffers 的官方文档网站本身也通过 GitHub Pages 服务托管。[[entities/google|google]] 公司也与 GitHub 有着密切的合作关系，其许多开源项目（包括 protoc 编译器）均发布在 GitHub 上。
